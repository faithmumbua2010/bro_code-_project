# Bro Code Python project
A collection of Python exercises and mini- projects from the Bro Code course 
covering fundamentals like variables,functions, operators and more.

## Contents
- 'madlibs.py' -  word game where you create a story by filling blanks with words
- 'weight_converter.py'- converts kgs to pounds and vice versa
- 'calculator.py' - I made a basic python calculator
- 'exercise_1.py' - calculating the area of a rectangle
- 'exercise_2.py' - Shopping cart program
- 'exercise_3.py'- calculating the circumference of a circle
- 'exercise_4.py' - calculating the area of a circle
- 'exercise_5.py' - finding the hypotenus of a triangle
- 'typecasting.py' - proces of converting a variable from one data type to another
- 'variables.py'- A variable is a container for storing data values(string, integer, float, boolean).
- 'input.py'-  A function that prompts the user to enter data . Returns the entered data as a string
